Titanic Data Analysis Project
Project Overview

This project focuses on analyzing the Titanic dataset to understand passenger survival patterns during the Titanic disaster.
The analysis explores how different factors such as gender, age, passenger class, and fare affected survival rates.

The project was performed using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn for data cleaning, exploratory data analysis (EDA),
and data visualization.

Objectives
Understand the Titanic dataset
Perform data cleaning and preprocessing
Handle missing values and outliers
Analyze passenger survival rates
Compare survival based on:
Gender, Age, Passenger Class, Fare

Create meaningful visualizations
Generate insights through data storytelling
Technologies Used
Python
Pandas
NumPy

Seaborn
Jupyter Notebook

Dataset Features

The dataset contains information about Titanic passengers, including:

Column	Description
PassengerId	Unique passenger ID
Survived	Survival status (0 = No, 1 = Yes)
Pclass	Passenger class
Name	Passenger name
Sex	Gender
Age	Passenger age
SibSp	Number of siblings/spouses aboard
Parch	Number of parents/children aboard
Ticket	Ticket number
Fare	Ticket fare
Cabin	Cabin number
Embarked	Port of embarkation

 Key Insights
Most passengers did not survive the Titanic disaster.
Female passengers had a higher survival rate than male passengers.
Children were more likely to survive.
First-class passengers had the highest survival rate.
Passengers who paid higher fares were more likely to survive.
Outliers were detected and handled using the IQR method.
